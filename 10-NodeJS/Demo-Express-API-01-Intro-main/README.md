# Demo d'un serveur API avec Express JS

## Avant de démarrer - Installation des dépendences
```
npm install
```

## Demarrer le serveur API
- Production
```
npm start
```
- Developpement
```
npm run dev
```

### Les dépendences du projet
- express
- cors
- yup
- morgan
- dotenv-flow
- cross-env
- nodemon _(save-dev)_
